Guide to Rayyan: Systematic Review Screening Tool

Rayyan is a free web application designed to help researchers screen articles for systematic reviews collaboratively and efficiently.

1. Access and Login/Account Creation

Access the website:

https://www.rayyan.ai/

  • If you do not have an account, click "Start For Free" in the upper right corner to create one.
  • If you already have an account (or after creating one), click "Login."

2. Create a New Review

Click "Create Review" in the upper right corner of the dashboard.

Required Settings:

  • Review Title: Enter a title (you can change this later).
  • Review Type: Select Systematic Review.
  • Review Domain: Select Biomedical.
  • Description: Enter a brief description (you can change this later).

3. Invite Collaborators

After creating the review, you can invite team members to participate in the screening process.

Invitation Fields:

  • User Email: Enter the collaborator's email address (e.g., `xxx@xxx.ac.jp`).
  • User Role: Select Reviewer.
  • Reason/Message: Not required for basic invitation.

The invited members will receive an email to join the review.

4. Import References

Once the review is created, you can import the citation files (e.g., RIS, CSV, PubMed format) exported from databases like PubMed, Scopus, and Cochrane.

Click "Add" or "Import" within the new review dashboard and upload the files one by one.

5. Screening Process

After importing references, you can start the screening process.

Screening Steps:

  • Title and Abstract Screening: Reviewers can screen articles based on titles and abstracts. They can label articles as "Include," "Exclude," or "Maybe."
  • Full-Text Screening: For articles marked as "Include" or "Maybe," reviewers can upload and review the full text to make final inclusion decisions.
  • Conflict Resolution: If there are disagreements between reviewers, Rayyan provides tools to resolve conflicts and reach consensus.

6. Exporting Results

After completing the screening, you can export the results for further analysis. Click on the "Export" button in the review dashboard to download the screened articles in various formats (e.g., RIS, CSV).

This guide was created to help users understand the basic process of setting up a screening project in Rayyan.
