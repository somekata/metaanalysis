How to Use

  1. 1
    Download template.txt
    Copy the blank template for each new paper you extract.
  2. 2
    Fill in the 10 sections
    Follow the structured format. Section 6 (Meta-analysis Raw Data) is the most critical — enter event.e, n.e, event.c, n.c for the 2×2 table.
  3. 3
    Load into SR Viewer
    Open sr_viewer.html, then drag & drop one or more .txt files. The viewer parses them automatically.
  4. 4
    Export for R / metafor
    Use the Export menu to download CSV or JSON. event.e / n.e / event.c / n.c columns map directly to metabin().

Extraction Format

# Section Key fields
1Study CharacteristicsAuthor, Year, Journal, Country, Design, Setting
2PopulationSample size, Age, Sex, Inclusion / Exclusion
3Exposure / ComparisonExposure, Comparison group, Follow-up period
4OutcomesPrimary outcome, Secondary outcomes, Definitions
5ResultsEffect size (OR/RR/HR), 95% CI, p-value, Model type
6Meta-analysis Raw Data2×2 table: event.e / n.e / event.c / n.c
7Risk of Bias (NOS)Selection, Comparability, Outcome
8ConfoundingKey confounders, Methods
9GRADERisk of bias, Inconsistency, Indirectness, Imprecision
10Final AssessmentSuitable for meta-analysis, Notes
