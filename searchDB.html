Search Strategy Example

Below is an example of search strategies used in databases like PubMed, Scopus, and Cochrane Library for a systematic review on treatments for carbapenem-resistant Pseudomonas aeruginosa infections.

*Note: This is an example and not an exhaustive or perfect search strategy.

# Search Query
1 "Pseudomonas aeruginosa"
2 "Carbapenem-resistant"
3 "Cefiderocol"
4 "Ceftazidime-Avibactum"
5 "Imipenem-Cilastatin-Relebactam"
6 #1 AND #2
7 #4 OR #5
8 #6 AND #3 AND #7

PubMed (MEDLINE)

  1. Go to: https://pubmed.ncbi.nlm.nih.gov/
  2. Select "Advanced Search."
  3. Enter #1 into the Query box and click "Add to History."
  4. Repeat the process for search terms #2 through #8, adding them all to History.
  5. Combine the history numbers (e.g., #8 AND #9) using Boolean operators as needed.
  6. Click on the total number of results and review the displayed citations.
  7. Click the "Save" button.
    Select "All results" and choose "PubMed" format.
    Click "Create File".

To increase the specificity of your search, consider limiting your keywords to the Title and Abstract fields.

Example: "Pseudomonas aeruginosa"[Title/Abstract]
Alternatively, when using the Advanced Search Builder, you can select the desired field (e.g., Title/Abstract) before adding the term to the Query box and history.

Scopus

  1. Go to: https://www.scopus.com/
  2. Select the "Advanced" search mode.
  3. Enter the combined search query (e.g., the full string for #8: `("Pseudomonas aeruginosa" AND "Carbapenem-resistant") AND "Cefiderocol" AND ("Ceftazidime-Avibactum" OR "Imipenem-Cilastatin-Relebactam")`) directly into the search box.
  4. Ensure the search fields include "Article title, Abstract, Keywords" for comprehensive results.
  5. Run the search and review the results.
  6. Refine results using side filters (e.g., "Article," "Clinical Trial").
  7. Select the desired citations and click "Export."
  8. Select "RIS format" for import into citation management software (e.g., EndNote, Zotero).
    In the Export window, ensure the following fields are selected for a complete systematic review record:
    - Citation Information (書誌情報)
    - Bibliographical information (目録情報), ensuring PubMed ID is selected.
    - Abstract & Keywords (抄録とキーワード)

*Note: When constructing advanced queries in Scopus, you can use field codes and specific operators in the query box.
Example: To limit a search to the title and abstract fields, you can use TITLE-ABS-KEY("Pseudomonas aeruginosa").

Cochrane Library

  1. Go to: https://www.cochranelibrary.com/
  2. Select "Advanced Search."
  3. Navigate to the "Search Manager."
  4. Enter search term #1 and click "Continue."
  5. Repeat the process for search terms #2 through #8, entering and combining them as necessary.
  6. Click on the search result count to display the citations.
  7. Select all or desired results and choose "Export selected citation(s)."
  8. Select "RIS (EndNote)" format, check the box to "include abstract," and click "Download."

*Note: The Cochrane Library offers advanced options for focused searching:
You can refine your search by Title, Abstract, and Keywords (S), or by using MeSH terms (controlled vocabulary) to improve specificity.
You can also use the Limits function to filter results based on various criteria, such as Content Type (e.g., "Cochrane Reviews," "Trials").
